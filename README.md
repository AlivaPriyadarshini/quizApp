# Quiz Application

A front-end Quiz Application built using HTML, CSS, and JavaScript.

## Features
- User Registration
- User Login
- LocalStorage-based Authentication
- Protected Dashboard
- Quiz Questions One by One
- Score Calculation
- Result Page
- Responsive Design

## Technologies Used
- HTML
- CSS
- JavaScript
- LocalStorage

## Folder Structure
quiz-app/
- index.html
- register.html
- dashboard.html
- quiz.html
- result.html
- css/
- js/

## How to Run
1. Download or create the project folder
2. Add all files properly
3. Open `index.html` in your browser
4. Register a new account
5. Login and start quiz
